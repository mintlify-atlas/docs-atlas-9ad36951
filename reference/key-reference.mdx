<Note>
Key notation follows standard Emacs conventions: `C-` means Control, `M-` means Meta (usually Alt or Escape), and `RET` means Return/Enter. Keys shown as `{C-h h}` in Hyperbole documentation are brace-delimited and can be pressed directly with the Action Key if Hyperbole is active.
</Note>

## Global keys

These keys are available whenever `hyperbole-mode` is active.

| Key | Description |
|-----|-------------|
| `M-RET` | Action Key — context-sensitive action in the current buffer |
| `C-u M-RET` | Describe what the Action Key will do without activating it |
| `M-<mouse-2>` | Action Mouse Key — same as Action Key, triggered by mouse |
| `C-h h` | Open the Hyperbole main menu in the minibuffer |
| `M-x hyperbole RET` | Open the Hyperbole main menu |
| `C-c \` | Open HyControl window control (if not previously bound) |

## Hyperbole main menu

Press `C-h h` to open the Hyperbole minibuffer menu, then type the first letter of each item to navigate. Press `q` to quit.

| Key | Menu item | Description |
|-----|-----------|-------------|
| `b` | Buttons | Button operations submenu |
| `d` | Demo | Interactive demonstrations |
| `f` | Find | Find/web search submenu |
| `k` | Koutliner | Koutliner operations |
| `m` | Msg | Toggle messaging button support |
| `r` | Rolo | HyRolo contact manager |
| `s` | Screen | HyControl screen/window/frame control |
| `u` | UsersGuide | Browse the Hyperbole Info manual |
| `q` | quit | Quit the menu |

### Demo submenu (`C-h h d`)

| Key sequence | Description |
|--------------|-------------|
| `C-h h d d` | Run the interactive Hyperbole button capabilities demo |
| `C-h h d i` | Browse the Hyperbole Info manual |
| `C-h h d k` | Run the interactive Koutliner demo |
| `C-h h d v` | Display the installed Hyperbole version |

### Screen submenu (`C-h h s`)

| Key sequence | Description |
|--------------|-------------|
| `C-h h s w` | Start HyControl window control mode |
| `C-h h s f` | Start HyControl frame control mode |

## Button operations

Access the button submenu with `C-h h b`.

| Key sequence | Description |
|--------------|-------------|
| `C-h h b c` | Create an explicit button |
| `C-h h b e` | Edit an existing explicit button |
| `C-h h b d` | Delete an explicit button |
| `C-h h b r` | Rename an explicit button |
| `C-h h b s` | Search for a button by name |
| `C-h h b g` | Jump to a global button |

### Mouse button creation

You can also create explicit buttons by dragging with the mouse:

| Drag operation | Result |
|----------------|--------|
| Drag between two windows (anywhere to anywhere) | Creates an implicit button link (`ibut`) |
| Drag from a button/buffer/file menu item between windows | Creates an explicit button link (`ebut`) |

## HyControl window mode

## HyRolo

Start HyRolo with `C-h h r` from the Hyperbole menu.

| Key | Description |
|-----|-------------|
| `C-h h r a` | Add a new HyRolo entry |
| `C-h h r e` | Edit the last matched entry |
| `C-h h r f` | Find (search) entries by regexp |
| `C-h h r k` | Kill (delete) an entry |
| `C-h h r m` | Mail to the e-mail address in an entry |
| `C-h h r n` | Move to the next entry in match results |
| `C-h h r p` | Move to the previous entry in match results |
| `C-h h r s` | Sort entries |
| `C-h h r t` | Display table of contents |
| `M-RET` | In a HyRolo match buffer, edit an entry or send mail |

## Koutliner

The Koutliner is an autonumbered hierarchical outliner. Enter it via `C-h h k` or by opening a `.kotl` file.

<Tabs>
  <Tab title="Navigation">

| Key | Description |
|-----|-------------|
| `C-n` | Move to next visible cell |
| `C-p` | Move to previous visible cell |
| `M-f` | Move forward one cell at the same level |
| `M-b` | Move backward one cell at the same level |
| `C-c C-n` | Move to next cell at any level |
| `C-c C-p` | Move to previous cell at any level |
| `M-RET` (on cell) | Collapse or expand cell and subtree |
| `M-RET` (left of cell) | Create a klink to this cell |

  </Tab>
  <Tab title="Editing">

| Key | Description |
|-----|-------------|
| `C-c C-a` | Add a sibling cell after the current cell |
| `C-c C-b` | Add a child (branch) cell below the current cell |
| `C-c C-k` | Kill (cut) the current tree |
| `C-c C-y` | Yank (paste) a killed tree |
| `M-<up>` | Move the current tree up past the previous sibling |
| `M-<down>` | Move the current tree down past the next sibling |
| `M-<left>` | Promote the current tree (decrease indent level) |
| `M-<right>` | Demote the current tree (increase indent level) |
| `C-c C-l` | Add a label/property to the current cell |

  </Tab>
  <Tab title="View control">

| Key | Description |
|-----|-------------|
| `C-c C-o` | Overview: show first line of each cell |
| `C-c C-e` | Expand all cells fully |
| `C-c <` | Show only cells up to the given level |
| `C-c >` | Show cells expanded to the given level |
| `C-c C-v` | Set the current view specification |

  </Tab>
</Tabs>

<Info>
The Koutliner demo is available at `C-h h d k`. It walks you through creating cells, moving trees, and setting view specifications interactively.
</Info>
