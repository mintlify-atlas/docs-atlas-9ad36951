GNU Hyperbole is a powerful, programmable hypertextual information management system built as a GNU Emacs package. It transforms your text into an interconnected web of smart hyperlinks with a single key press — no markup language required. Load it once, and your buffers come to life.

## What is Hyperbole?

Hyperbole automatically recognizes dozens of common patterns in any buffer regardless of mode and activates them as hyperbuttons with a single key. Email addresses, URLs, grep output, git commits, programming backtraces, Org links, Info cross-references, and more all become instantly clickable — no markup required.

## Get started in 3 steps

<Steps>
  <Step title="Install via the Emacs Package Manager">
    Add these lines to your `~/.emacs` or `~/.emacs.d/init.el`:

    ```emacs-lisp
    (require 'package)
    (unless (package-installed-p 'hyperbole)
      (package-refresh-contents)
      (package-install 'hyperbole))
    (hyperbole-mode 1)
    ```

    Restart Emacs and Hyperbole will download, compile, and activate automatically.
  </Step>
  <Step title="Open the interactive demo">
    Press `C-h h d d` to launch the interactive DEMO file, which walks you through Hyperbole's button capabilities hands-on. This is the fastest way to learn.
  </Step>
  <Step title="Press the Action Key">
    Move point over any URL, file path, email address, or other recognizable pattern in any buffer and press `M-RET` (the Action Key). Hyperbole activates it automatically.
  </Step>
</Steps>

<Note>
  Hyperbole works with GNU Emacs 28 or above and requires no external dependencies beyond standard Emacs libraries.
</Note>
